Built for real pipelines

One assistant across your whole scene

From blockout to look-dev — talk to Blender in English or Persian, with tools that respect undo.

  • Multi-provider brain

    Ollama, OpenAI, Claude, DeepSeek, Qwen, GLM, and any OpenAI-compatible endpoint.

  • Domain skills

    Modeling, materials, lighting, sculpt assist, review — extend with YAML.

  • Private by default

    Keys in the OS keyring. Local Only mode. Scene context stays under your control.

  • MCP for your AI tools

    Drive the same open Blender scene from Cursor, Claude Desktop, Claude Code, Windsurf, VS Code, and more.

Three steps

From install to first mesh

  1. 01

    Run the installer

    Windows, macOS, or Linux — finds Blender and sets up the sidecar.

  2. 02

    Connect a model

    Open Providers, test Ollama or a cloud key, set your default.

  3. 03

    Chat in the N-Panel

    Press N → BlenderAI → describe what you need in the scene.
