Built for real pipelines

One assistant across your whole scene

From blockout to look-dev — talk to Blender in English or Persian, with tools that respect undo.

  • Multi-provider brain

    Ollama, OpenAI, Claude, DeepSeek, Qwen, GLM, and any OpenAI-compatible endpoint — configured in a graphical Providers UI.

  • Domain skills

    Modeling, materials, lighting, sculpt assist, scene organize, render tune, and more — extend with YAML.

  • Private by default

    Keys in the OS keyring. Local Only mode. Scene context stays under your control.

  • Fast WebUI

    Streaming chat, polished selects and toggles, EN/FA with independent LTR/RTL direction.

Three steps

From install to first mesh

  1. 01

    Run the installer

    Windows, macOS, or Linux — finds Blender and sets up the sidecar.

  2. 02

    Connect a model

    Open Providers, test Ollama or a cloud key, set your default.

  3. 03

    Chat in the N-Panel

    Open Chat from BlenderAI and describe what you need in the scene.

MCP

Drive Blender from Cursor
