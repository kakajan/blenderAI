# Documentation & GitHub Pages

**Live site:** [kakajan.github.io/blenderAI](https://kakajan.github.io/blenderAI/)

Landing files in this folder: `index.html`, `landing.css`, `landing.js`.

The landing page loads live repository stats from the [GitHub API](https://api.github.com/repos/kakajan/blenderAI) (stars, forks, language, license, last update) and supports **English (default)** ↔ **فارسی** via the FA/EN toggle.

## Enable GitHub Pages

1. Open [repository Settings → Pages](https://github.com/kakajan/blenderAI/settings/pages)
2. Source: **Deploy from a branch**
3. Branch: `main` · folder: `/docs`
4. Save — the site will be at `https://kakajan.github.io/blenderAI/`

Optional: set repository **Homepage** to that URL under Settings → General.
