# Documentation & GitHub Pages

**Live site (after enabling Pages):** [kakajan.github.io/blenderAI](https://kakajan.github.io/blenderAI/)

Landing files: `index.html`, `landing.css`, `landing.js` in this folder.

## Enable GitHub Pages

1. Open [repository Settings → Pages](https://github.com/kakajan/blenderAI/settings/pages)
2. Source: **Deploy from a branch**
3. Branch: `main` · folder: `/docs`
4. Save — the landing will be at `https://kakajan.github.io/blenderAI/`
